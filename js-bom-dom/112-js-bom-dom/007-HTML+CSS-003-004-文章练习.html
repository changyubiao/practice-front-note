长城全新高端皮卡官图曝光 上海车展首发

2019-04-02 15:20:27 新浪汽车 评论中大奖11人参与 收藏本文

新浪汽车讯 长城全新高端皮卡官图曝光，据悉其将在上海车展正式进行旗下高端品牌的发布，并带来一款全新豪华皮卡车型越野版，将标配三把锁、涉水喉和绞盘。新车将比目前在售的长城皮卡车型价格贵、尺寸更大，车型尺寸与福特猛禽对标，但是售价将更亲民。

曝光的官图为新车的越野版，新车整体样式更加霸气，采用了全新的前脸，梯形的进气格栅采用网状的装饰风格，占据了前脸的大部分，前侧的大灯看上去也非常犀利，前保险杠采用一体化的设计，让整体前脸显得很饱满。加装的涉水喉排气，凸出的轮眉包围，强化了越野特性。同时，其还使用了定制的前后杠，前杠带有绞盘孔，后杠带有拖车钩，车顶还有行李架等。

根据此前曝光的专利图，这款长城全新皮卡车型可能包含了长/短货箱版，还有单排版、改装版等多款车型，将是一个全新的产品序列，而非单款产品。而此前有媒体猜测，这次的新产品可能是长城皮卡全新打造的风骏8和风骏9，不过这些猜测始终没有得到长城皮卡官方的证实。

这款长城全新皮卡车型基于全新平台打造，并非此前报道的H9平台，并且其将在重庆永川新工厂生产制造。据了解，长城汽车重庆新工厂未来主要生产长城皮卡和哈弗品牌SUV产品，预计2019年年底建成投产，可为长城汽车新增整车及配套零部件16万台套。
